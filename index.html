Projects

Below is a few of my recent personal/graduate school/competition projects.

Automatic Sentiment Analysis App for YouTube Comments

The app takes in your YouTube video link as input, pull comments data from YouTube API, pre-process it for sentiment analysis (labelling if a comment is positive, neutral, or negative), then the processed data is visualized via the Gradio interface.

Exploring Data Visualization: A Tutorial Using Thailand’s Tourism Industry Data

Analyzed and Extracted Insights from 3 Years of Tourism Data using Python. Utilized geopandas library to visualize these average yearly tourism earnings from 2019 to 2022 on Thailand’s Map. Condensed 4 years’ worth of Thailand’s Tourism data into easy to digest insights.

QuickEcon - Your Handy Economics Evaluation Tool

Built Net Present Value and Internal Rate of Return Calculator using Numpy, Matplotlib, and Streamlit Libraries in Python.

Machine Learning Approach for Anomaly Detection in Process Industry (On-going)

This project aims to close the traditional monitoring gap by developing unsupervised ML model/algorithm which can detect abnormalities in the industrial processes and notify the user for prompt decision making.

Building an FAQs AI Assistant for Kaggle Platform (On-going)

Create dataset from unstructured data on the platform, utilize PEFT for fine-tuning open-source LLM models with the training data, and evaluate the AI assistant performance on the training/validation dataset.
